# To-Do List Full Stack App

Full stack to-do list web app.

# Technoloy used:
EJS, Express.js, Node.js, MongoDB

## Authors

- [@kriswen](https://www.github.com/kris-wen)

## Deployment

To install dependencies

```bash
  npm install
```

To start this project run

```bash
  node app.js
```
